# ProyectoOdoo

Este proyecto consiste en la implantación y configuración completa de un sistema ERP utilizando Odoo. El caso práctico simula la digitalización total de Renova Studio, una empresa ficticia dedicada a las reformas integrales y el diseño de interiores.

El objetivo principal ha sido conectar el entorno web (Front-end) con la gestión administrativa, de ventas y facturación (Back-end), garantizando que todo el flujo de trabajo esté automatizado y cumpla con la normativa fiscal española.
